## 简介

MST最小生成树主页只是一个名字，本质上是一个便捷搜索网站的简洁网页。

实现方法都是非常基础的js代码，没有用到什么框架。

## 功能

保留原来的多站点搜索能力，默认支持百度、Google、Bing、知乎、豆瓣、Bilibili、微信和微博。

新增个人设置页 `settings.html`，可以在浏览器本地编辑并保存：

- 主页标题、副标题、搜索框提示、背景和强调色
- 默认搜索网站
- 搜索网站列表、搜索地址和启用状态
- 左上角快捷入口
- 配置导入/导出

配置保存在浏览器 `localStorage` 中，不需要后端服务。
